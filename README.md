## 🗂️ Tasks Overview

### 🌸 Task 1 — Iris Flower Classification
> Classify Iris flowers into 3 species using machine learning

| Property | Details |
|---|---|
| Dataset | Iris.csv — 150 samples, 4 features |
| Models | KNN · Decision Tree · Logistic Regression · Random Forest |
| Best Accuracy | **100%** (Random Forest) |
| Folder | `Task1_Iris_Flower_Classification/` |

**Key Highlights:**
- Full Exploratory Data Analysis with pairplots, heatmaps, violin plots
- Compared 4 ML models with accuracy + 5-fold cross-validation
- Feature importance, confusion matrices, decision tree visualization
- Optimal K selection for KNN

---

### 📊 Task 2 — Unemployment Analysis
> Analyze unemployment trends across Indian states and regions

| Property | Details |
|---|---|
| Dataset | Unemployment_in_India.csv (768 records) + Unemployment_Rate_upto_11_2020.csv (267 records) |
| Period | 2019 – 2020 |
| Focus | COVID-19 impact, Rural vs Urban, Region-wise comparison |
| Folder | `Task2_Unemployment_Analysis/` |

**Key Highlights:**
- National unemployment trend with COVID-19 period highlighted
- Rural vs Urban comparison over time
- State-wise top/bottom 10 unemployment chart
- COVID-19 impact analysis (Pre / During / Post)
- Labour participation vs unemployment scatter plot

## 🚀 How to Run

### ▶ Google Colab (Recommended)
1. Open [colab.research.google.com](https://colab.research.google.com)
2. Upload the `.ipynb` notebook for the task you want to run
3. Click **Runtime → Run All**
4. Upload the CSV file(s) when the upload button appears
5. Everything runs automatically!

### ▶ Local Machine
```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/codealpha_tasks.git
cd codealpha_tasks

# Install dependencies
pip install -r requirements.txt

# Run Task 1
jupyter notebook Task1_Iris_Flower_Classification/CodeAlpha_IrisFlowerClassification.ipynb

# Run Task 2
jupyter notebook Task2_Unemployment_Analysis/CodeAlpha_UnemploymentAnalysis.ipynb
```
